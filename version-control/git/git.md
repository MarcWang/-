

### 設定 name 和 email
``` sh
git config --global user.name "Jimmy Kuo"
git config --global user.email "jimmy@gogojimmy.net"
```

### 增加色彩
``` sh
git config --global color.ui true
```

### 查詢設定
``` sh
git config --list
```

### ㄧ次增加一個檔案
``` sh
git add readme.md 
```

### ㄧ次增加多個檔案
``` sh
git add .
```

git add -i 


### Commit
``` sh
git commit
git commit -m 'message'

(Shift + z,z) -> save
```

### Push
``` sh
git push origin master
```
